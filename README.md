#Hate-Speech-Detection-using-NLP-and-LSTM-
1.Developed a Natural Language Processing model to detect hate speech in tweets. 
2.Performed data preprocessing including cleaning text, removing symbols, stopwords removal, and Lemmatization using Spacy. 
3.Converted text data into numerical vectors using One-Hot Encoding and Embedding layers.  
4.Handled imbalanced dataset using SMOTE .
